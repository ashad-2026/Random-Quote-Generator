## **Описание**

*Random Quote Generator — это GUI‑приложение для генерации случайных цитат с возможностью фильтрации по автору и теме. Программа отображает историю последних 20 сгенерированных цитат, позволяет сохранять все данные в JSON‑файл и очищать историю. Решает задачу быстрого доступа к вдохновляющим высказываниям и помогает изучать цитаты разных авторов и тематик.*

## Требования для запуска

Что нужно установить на компьютер: 
* Python 3.6 или выше 
* Библиотеки: `tkinter`, `json`, `random`, `os`
  
## Как запустить
```bash 
git clone https://github.com/ashad-2026/Random-Quote-Generator
cd Random-Quote-Generator
python main.py
```
## Как пользоваться

1.  При запуске откроется окно приложения с кнопкой «Сгенерировать цитату» и полями для фильтров.
2.  Нажмите кнопку **«Сгенерировать цитату»** — в окне отобразится случайная цитата с указанием автора и темы.
3.  Используйте **фильтры**:
    -   выберите автора из выпадающего списка «Фильтр по автору»;
    -   выберите тему из выпадающего списка «Фильтр по теме»;
    -   после установки фильтров новые генерируемые цитаты будут соответствовать выбранным критериям. 
4.  История последних 20 цитат отображается в нижней части окна в области «История цитат».
5.  Для **очистки истории** нажмите кнопку «Очистить историю».
6.  Для **сохранения данных** (цитат и истории) нажмите кнопку «Сохранить данные» — информация запишется в файл `quotes_data.json`.
  
## Пример работы

**Сценарий 1.  "Генерация случайной цитаты"**
1.  Запустите приложение.
2.  Нажмите **«Сгенерировать цитату»**.    
3.  В окне появляется:
    ```
    "Знание — сила"
    — Фрэнсис Бэкон
    Тема: Мудрость
    ```  
4.  Цитата добавляется в историю внизу окна.
  
**Сценарий 2. "Фильтрация по автору"**
5.  В фильтре «Фильтр по автору» выберите «Сократ».
6.  Нажмите **«Сгенерировать цитату»**.  
7.  Если есть цитаты Сократа, отобразится одна из них, например:
    ```
    "Познай самого себя"
    — Сократ
    Тема: Самопознание
    ```
**Сценарий 3. "Сохранение данных"**
8.  Сгенерируйте несколько цитат.  
9.  Нажмите **«Сохранить данные»**. 
10.  Появляется сообщение: «Данные сохранены в quotes_data.json».   
11.  Закройте и перезапустите приложение — история и список цитат восстановятся из файла.
 
**Обработка ошибок:**
-   Если по фильтрам цитат не найдено → сообщение: «По заданным фильтрам цитат не найдено!»
